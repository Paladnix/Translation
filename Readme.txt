此份翻译全部来自网页
https://www.microsoft.com/cognitive-services/
我们翻译了网页内与智能相关的一系列API

网页内的每份API reference内都有示例代码
由于篇幅的原因，我们没有将示例代码放入译文内
示例代码的语言包括Curl、C#、Java、JavaScript、ObjC、PHP、Python、Ruby
示例代码是很好的模板，会为大家的编程带来很大的便利。请务必仔细阅读。

每次调用API都会有相应的response
由于篇幅的原因，我们只把调用成功时的response放在译文内。
当调用失败时，response往往包括错误码及一条错误信息
请在官网的API reference处根据相应错误码解决错误

我们在译文内省略了Request headers
它包括Content-Type和Ocp-Apim-Subscription-Key
其中Content-Type是可选的，它指定request body中的Media type
Ocp-Apim-Subscription-Key是必填的，它是使用该份API的凭证（查看你的 Cognitive Services accounts）
